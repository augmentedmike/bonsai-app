## Pitch Framework for This Audience

**One-liner:** Bonsai is the AI-native Kanban board that doesn't just track your work — it executes it.

**The hook for angels:** "Every developer tool from 2010–2020 automated tracking. We're the first to automate doing. Our AI agents turn a ticket into a shipped PR."

**The proof points to have ready:**
- Daily active developers using it (even 10 real users > 1,000 signups)
- Number of tickets shipped autonomously per week
- Time-to-ship comparison: human vs. Bonsai agent

**The investor narrative:** Linear proved beautiful project management. Cursor proved AI coding. Bonsai combines them — you plan in natural language, agents ship the code.
